**viber3D** - a modern starter kit for 3D browser games.

---

## ⚠️ Important Note

**Do not install this package directly!** Instead, use the CLI tool to create a new project:

```bash
npx viber3d@latest my-project
```

This ensures you get the complete starter kit with all necessary configurations and dependencies properly set up.
